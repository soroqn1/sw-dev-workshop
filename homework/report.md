# Звіт про виконання завдання: Багатопотоковість у Java

## Завдання
Розробити програму, яка одночасно виводить цифри з різними затримками:
1. Цифру "1" із затримкою в 1 секунду.
2. Цифру "2" із затримкою у 2 секунди.
3. Цифру "3" із затримкою у 3 секунди.

## Опис реалізації
Для реалізації завдання була обрана мова програмування Java та механізм потоків (`Threads`). Оскільки вивід має відбуватися паралельно, кожне завдання було винесено в окремий потік.

### Основні компоненти:
- **`Thread`**: Клас для створення паралельних процесів.
- **`Thread.sleep(ms)`**: Метод для зупинки потоку на заданий час (мілісекунди).
- **Лямбда-вирази**: Використані для лаконічного опису логіки кожного потоку.

## Код програми
```java
public class homework {
    public static void main(String[] args) {
        Thread thread1 = new Thread(() -> {
            while (true) {
                try {
                    Thread.sleep(1000);
                    System.out.println("1");
                } catch (InterruptedException e) {
                    break;
                }
            }
        });

        Thread thread2 = new Thread(() -> {
            while (true) {
                try {
                    Thread.sleep(2000);
                    System.out.println("2");
                } catch (InterruptedException e) {
                    break;
                }
            }
        });

        Thread thread3 = new Thread(() -> {
            while (true) {
                try {
                    Thread.sleep(3000);
                    System.out.println("3");
                } catch (InterruptedException e) {
                    break;
                }
            }
        });

        thread1.start();
        thread2.start();
        thread3.start();
    }
}
```

## Результат роботи
При запуску програми в консолі спостерігається такий вивід (послідовність залежить від часу запуску потоків):
- Кожну 1 секунду: `1`
- Кожні 2 секунди: `2`
- Кожні 3 секунди: `3`

Приклад виводу за перші 6 секунд:
```text
1
2
1
3
1
2
1
1
3
2
1
...
```
